# **App Name**: TrustLink

## Core Features:

- Worker Profile Management: Workers can create and manage their personal and professional profiles, including name, trade, skills, profile picture, and a unique, verifiable QR code.
- Job Logging: Informal workers can log completed jobs, providing details such as description, date of completion, and an optional photo of the finished work.
- Client Job Verification: Clients can verify job completion through a simple process, such as scanning a QR code or receiving an SMS link, updating the job status automatically.
- Dynamic Trust Score: A real-time scoring system that calculates and updates a worker's trust score based on successfully verified jobs and client ratings.
- Shareable Public Reputation Page: Each worker receives a unique, shareable link to a public reputation page displaying their profile, verified jobs, and current trust score.
- Notification System: Workers receive alerts when their jobs are successfully verified by a client or when they receive a new rating, ensuring they stay informed.
- Admin Dashboard: A minimal administrative interface to manage platform users, review reported jobs, and ensure the overall health of the TrustLink community.

## Style Guidelines:

- Primary color: Deep teal (#00796B), conveying trust and a modern tech feel. HSL: H:173, S:100%, L:24%.
- Background color: Soft white (#FDFDFD), providing clarity and a clean, unobtrusive canvas. HSL: H:0, S:0%, L:99%.
- Accent color: Golden yellow (#FFC107), used to highlight achievement, ratings, and key interactive elements. HSL: H:43, S:100%, L:51%.
- Body and headline font: 'Inter', a neutral and highly readable sans-serif that embodies a modern, clean aesthetic.
- Utilize clear, vector-based icons that are universally understandable and align with the app's clean, modern design language, particularly for job categories and verification steps.
- A responsive, mobile-first layout prioritizing essential information and user actions, ensuring low-friction onboarding and accessibility across all device types, consistent with PWA principles.
- Incorporate subtle, functional animations for transitions and feedback, enhancing the user experience without being distracting, particularly during job verification or profile updates.
